# NutriBite 

### Home Page
![Home Page](nutribite-health-hub-main/Screenshot%202026-06-22%113752.png)

NutriBite is a modern health-aware food delivery platform designed to help users make smarter food choices based on their personal nutrition goals. Instead of focusing only on food delivery, NutriBite provides detailed nutritional insights, health scores, and personalized meal recommendations to support healthier eating habits.

## Features

* Personalized recommendations based on health goals such as Weight Loss, Weight Gain, Muscle Building, and Healthy Lifestyle.
* Detailed nutrition information including Calories, Protein, Carbohydrates, Fat, Fiber, and Sugar.
* Health Score system to help users quickly evaluate meal quality.
* Smart filtering for Vegetarian, Vegan, High-Protein, Low-Carb, and other dietary preferences.
* Interactive Nutrition Dashboard to track daily and weekly nutrition intake.
* Restaurant and meal discovery with nutrition-focused search and filters.
* Favorites, Cart, Checkout, and Order Tracking functionality.
* Fully responsive and mobile-friendly user interface.

## Tech Stack

* React
* TypeScript
* Vite
* Tailwind CSS
* Shadcn UI
* React Router
* Recharts
* Local Storage

## Purpose

The goal of NutriBite is to bridge the gap between food delivery and healthy eating by helping users discover meals that align with their dietary preferences and fitness goals through a simple, intuitive, and modern web experience.
